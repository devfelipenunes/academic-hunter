What Is MCP and Why Does It Matter?

The Model Context Protocol (MCP) is an open standard that lets AI assistants connect directly to external tools — like giving Claude a research lab to command.

No more copy-paste

Instead of copying abstracts into a chat, your AI runs run_search across 7 databases in real time and returns structured results.

Autonomous exploration

Claude discovers jargon with quick_topic_discovery, configures anchors with update_config, then snowballs through citation graphs. All without you touching a config file.

Second brain integration

Findings land directly in your Obsidian vault via export_to_obsidian — with proper frontmatter, tags, and links. Your research feeds your knowledge base automatically.

Semantic memory

Every paper is indexed into ChromaDB. Later, you or your AI can ask ask_papers a question and get answers grounded in your own collected literature.

Works with any MCP client

Claude Desktop, Cursor, VS Code, Gemini, custom agents — anything that speaks MCP can drive Academic Hunter.

Also a Python CLI

Prefer the terminal? Run academic-hunter directly. The same engine powers both interfaces.

Behind the Scenes

When you or your AI triggers a search, here is what happens.

Configure

Research anchors and technical weights define what the engine looks for. Your AI can do this with update_config.

Mine

7 academic APIs are queried concurrently. Results are deduplicated by DOI and fuzzy title, then enriched with missing abstracts.

Score

Keyword + semantic screeners evaluate every paper. Only those above your threshold survive — noise is discarded.

Report

Results exported as CSV, BibTeX, RIS, and Markdown with PRISMA flow. Papers indexed into ChromaDB for future semantic queries.

Connect in 30 Seconds

Claude Desktop, Cursor, VS Code, or any MCP client.

{
  "mcpServers": {
    "academic-hunter": {
      "command": "/path/to/venv/bin/python",
      "args": ["-m", "academic_hunter.interfaces.mcp.server"],
      "env": { "PYTHONPATH": "/path/to/src" }
    }
  }
}

Or run python install.py — the auto-installer injects the config for you. 

# Clone and install
git clone https://github.com/devfelipenunes/academic-hunter.git
cd academic-hunter
python install.py

# Start the MCP server
academic-mcp

# Or use the CLI directly
academic-hunter --limit 10
